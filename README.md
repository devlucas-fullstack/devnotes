# DevNotes

**DevNotes** é uma aplicação web para criação, edição e organização de notas de desenvolvedor.  
O projeto foi construído com foco em simplicidade e desempenho, oferecendo uma experiência intuitiva para registrar ideias, trechos de código, lembretes e referências técnicas.

## Visão Geral

Este repositório contém o código-fonte da aplicação **DevNotes**, construída com **Vite**, **TypeScript**, **Tailwind CSS** e **HTML**. A aplicação permite ao usuário criar notas rápidas, organizá-las por categorias e salvar conteúdos técnicos locais ou remotamente (segundo futuras integrações).  

Atualmente o projeto é ideal para desenvolvedores que querem manter suas anotações de estudo ou trabalho organizadas em um único lugar.

---

## Funcionalidades

- Registrar e editar notas em Markdown ou texto simples
- Organização por categorias ou tags
- Interface responsiva e limpa
- Armazenamento local (até integração com backend no futuro)
- Otimizado com Vite e estilizado com Tailwind CSS
